In addition to receiving notifications of file transcoding results through the [event notification mechanism](https://intl.cloud.tencent.com/document/product/1041/33499), MPS users can also query details of a specified transcoding task through the DescribeTaskDetail API. This API is generally used to query the progress and results of a transcoding task that was manually initiated by the ProcessMedia API. Returned results can be `WAITING`, `PROCESSING`, or `FINISH`.
- **WAITING**: The task has been initiated and is waiting to be processed.
- **PROCESSING**: The task is being processed.
- **FINISH**: The task has been completed.
