### Anycastパブリックネットワーク加速AIAとは何ですか？
AIAは、ビジネスのパブリックネットワークアクセス体験を大幅に向上させることができるグローバルなDSAネットワークです。AIAは、他のアプリケーション層加速サービスと異なり、IP伝送の品質最適化とマルチポートによる最寄りアクセスを達成し、ネットワーク伝送のジッタ、パケットロスなどの問題を減らし、最終的にクラウド上のアプリケーションのサービス品質を向上させて、サービス範囲を拡大し、バックエンド配置を合理化します。

### Anycast Elastic IPアドレスとパブリックネットワークIPの違いは？
Anycast Elastic IPアドレスは多くの場所で同時に公開され、アクセストラフィックは最も近い地域に到達してからTencent CloudのBackbone Network専用回線に伝送ます。この製品の技術的な原理はAnycastなので、帯域幅の価格はより高く、ユーザーはこの点に注意する必要があります。
一般的なパブリックネットワークIPとEIPは1つの地域、つまりIPが属する地域だけで公開されます。通常の状態では、アクセストラフィックはパブリックネットワークを通して該当IPに到達します。

|  | Anycast Elastic IPアドレス | 非Anycast Elastic IPアドレス |
|---------|---------|---------|
| 公開方式 | Anycast | Unicast |
| IP公開地域 | 複数 | 唯一 |
|　ネットワーク価格　|　単価はより高くて、加速トラフィックによって課金　|　安くて、通常のパブリックネットワーク価格によって課金 |
| バインディング解除操作 | 対応可能 | 対応可能 |
| AIA加速　| AIA加速を提供	 |　デフォルトでパブリックネットワーク。パブリックネットワークに障害が発生したときに一時的な無料のトラフィックスケジューリングを提供 |

### なぜ地域（region）を区別しますか？
Tencent Cloudはさまざまな地域間で最大限の安定性とフォールトトレランスを確保するために、地域間がお互いに完全に分離される必要があります。

### 加速地域とは何ですか？
Anycast Elastic IPアドレスが公開された地域は加速地域です。

### サービス地域とは何ですか？
Anycast Elastic IPアドレスを作成する時、指定されたIPの地域がサービス地域となり、該当地域のリソースだけをバインディングできます。バックエンドロジック計算（CVMなど）はサービス地域で行われます。

### CVMのプライベートネットワークIPとパブリックネットワークIPの違いは何ですか？
プライベートネットワークIP：プライベートネットワークIPアドレスとも呼ばれ、ソースIPがプライベートネットワークのクライアントにサービスを提供する接続アドレスです。
パブリックネットワークIP：パブリックネットワークIPアドレスとも呼ばれ、ソースIPがパブリックネットワークのクライアントに対するパブリックネットワーク通信を行う接続アドレスです。
両者はネットワークアドレス変換を通して互いに直接マッピングすることができます。
同じ地域内のサーバーはプライベートネットワークを通して通信できますが、異なる地域のサーバーはパブリックネットワークしか利用できません。
### Anycastパブリックネットワーク加速の課金方法は？
Anycast Elastic IPアドレスを購入し、継続的な加速サービスを取得して、生成されたトラフィックは加速トラフィックであり、単価は通常のパブリックネットワークよりも高くなります。
AnycastパブリックネットワークIP以外の場合、パブリックネットワークに障害が発生した時、Tencent Cloudはユーザーに一時的な加速サービスを提供し、サービスの中断を回避できるようにします。これは無料の付加価値サービスで、ネットワーク料金は通常のパブリックネットワークの価格に従って課金されるので、ユーザーは注意する必要がありません。

### 加速トラフィックの課金方法は？
加速トラフィックは、Anycast Elastic IPアドレス（Anycast EIP）によって生成されたトラフィックで、1つのアカウントの合計加速トラフィック曲線が月95ピーク値課金により計算された後の値は決済帯域幅となります。

### 加速トラフィックの課金はアウトバウンド方向ですか、それともインバウンド方向ですか？
加速トラフィックはアウトバウンドとインバウンドのうち高い方向で課金されます。
 
### Anycast Elastic IPアドレス（Anycast EIP）はトラフィックに応じた従量制課金をサポートしますか？
サポートしません。Anycast Elastic IPアドレスで発生されたトラフィックは帯域幅によって課金されます。

### Anycast EIPはどのようにスピードを制限しますか？
Anycast EIPに帯域幅制限を設定する場合、設定の対象はすべての帯域幅ではなく、各地域のアウトバウンドの帯域幅制限です。
例えば、公開ドメインAのAnycast EIP帯域幅が100 Mbpsに設定されている場合、公開ドメインAの各公開地域の帯域幅は100 Mbpsに制限され、合計帯域幅は100 Mbpsを超えます。
