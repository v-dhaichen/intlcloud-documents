

## 機能説明

返された結果のエラーコードは、ユーザーがクラウドAPIを呼び出した結果を示します。codeが0の場合、呼び出しは成功します。それ以外の場合、呼び出しは失敗します。呼び出しが失敗した場合、ユーザーはパブリックエラーコードリストに基づいてエラーの原因を特定し、対処することができます。
例：

```
{
    "code": 5100
}
```
