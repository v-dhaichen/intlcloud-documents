

当您云服务器购买完成后，您可以在购买的服务器上搭建一个属于自己的网站或者论坛。

本文主要介绍在云服务器 CVM 搭建网站的几种方式，您可以根据使用需要来选择。

![](https://main.qcloudimg.com/raw/47d7f76597e7e9318a31ab72590692cb.png)

## 搭建方式
腾讯云针对主流的网站系统，提供了多种类型的建站教程。搭建方式为手工搭建，其特点为：


| | 手动搭建 |
|---------|---------|
| 搭建方式 | 以手动的方式安装所需软件，可定制化。 |
| 特点 |   配套版本可以灵活选择。 |
| 所需时间 | 比较长，需自行安装相关软件。 |
| 难易程度|  需要对软件配套版本及安装方法有一定的了解。 |
