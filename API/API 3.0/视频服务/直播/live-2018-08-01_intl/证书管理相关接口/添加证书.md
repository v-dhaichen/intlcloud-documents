## 1. 接口描述

接口请求域名： live.tencentcloudapi.com 。

添加证书

默认接口请求频率限制：200次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见[公共请求参数](/document/api/267/20459)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：CreateLiveCert |
| Version | 是 | String | 公共参数，本接口取值：2018-08-01 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| CertType | 是 | Integer | 证书类型。0-用户添加证书；1-腾讯云托管证书。 |
| HttpsCrt | 是 | String | 证书内容，即公钥。 |
| HttpsKey | 是 | String | 私钥。 |
| CertName | 否 | String | 证书名称。 |
| Description | 否 | String | 描述。 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| CertId | Integer | 证书ID|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 添加证书

#### 输入示例

```
https://live.tencentcloudapi.com/?Action=CreateLiveCert
&CertType=1
&CertName=name-crt
&HttpsCrt=XXXXX
&HttpsKey=YYYYYY
&Description=detail
&<公共请求参数>
```

#### 输出示例

```
{
    "Response": {
        "RequestId": "eac6b301-a322-493a-8e36-83b295459397"
    }
}
```
