## 1. 接口描述

接口请求域名： live.tencentcloudapi.com 。

获取单个回调模板

默认接口请求频率限制：500次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见[公共请求参数](/document/api/267/20459)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：DescribeLiveCallbackTemplate |
| Version | 是 | String | 公共参数，本接口取值：2018-08-01 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| TemplateId | 是 | Integer | 模板Id。 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| Template | [CallBackTemplateInfo](/document/api/267/20474#CallBackTemplateInfo) | 回调模板信息。|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|
