## 1. 接口描述

接口请求域名： live.tencentcloudapi.com 。

创建录制模板，成功返回模板id后，需要调用[CreateLiveRecordRule](/document/product/267/32615)接口，将模板id绑定到流进行使用。
<br>录制相关文档：[直播录制](/document/product/267/32739)。

默认接口请求频率限制：200次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见[公共请求参数](/document/api/267/20459)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：CreateLiveRecordTemplate |
| Version | 是 | String | 公共参数，本接口取值：2018-08-01 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| TemplateName | 是 | String | 模板名。非空的字符串 |
| Description | 否 | String | 描述信息。 |
| FlvParam | 否 | [RecordParam](/document/api/267/20474#RecordParam) | Flv录制参数，开启Flv录制时设置。 |
| HlsParam | 否 | [RecordParam](/document/api/267/20474#RecordParam) | Hls录制参数，开启hls录制时设置。 |
| Mp4Param | 否 | [RecordParam](/document/api/267/20474#RecordParam) | Mp4录制参数，开启Mp4录制时设置。 |
| AacParam | 否 | [RecordParam](/document/api/267/20474#RecordParam) | Aac录制参数，开启Aac录制时设置。 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| TemplateId | Integer | 模板Id。|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 请求示例

#### 输入示例

```
https://live.tencentcloudapi.com/?Action=CreateLiveRecordTemplate
&TemplateName=templat
&Description=test
&FlvParam.RecordType=0
&FlvParam.RecordInterval=1800
&FlvParam.StorageTime=600
&HlsParam.RecordType=0
&HlsParam.RecordInterval=1800
&HlsParam.StorageTime=600
&<公共请求参数>
```

#### 输出示例

```
{
  "Response": {
    "TemplateId": 1000,
    "RequestId": "3c140219-cfe9-470e-b241-907877d6fb03"
  }
}
```
