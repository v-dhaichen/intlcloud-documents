## 1. 接口描述

接口请求域名： vod.tencentcloudapi.com 。

* 仅当待删分类无子分类且无媒体关联情况下，可删除分类；
* 否则，请先执行[删除媒体](/document/product/266/31764)及子分类，再删除该分类；

默认接口请求频率限制：100次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见[公共请求参数](/document/api/266/31756)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：DeleteClass |
| Version | 是 | String | 公共参数，本接口取值：2018-07-17 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| ClassId | 是 | Integer | 分类 ID |
| SubAppId | 否 | Integer | 点播[子应用](/document/product/266/14574) ID。如果要访问子应用中的资源，则将该字段填写为子应用 ID；否则无需填写该字段。 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 删除视频分类

#### 输入示例

```
https://vod.tencentcloudapi.com/?Action=DeleteClass
&ClassId=1
&SubAppId=1
&<公共请求参数>
```

#### 输出示例

```
{
  "Response": {
    "RequestId": "6ca31e3a-6b8e-4b4e-9256-fdc700064ef3"
  }
}
```
