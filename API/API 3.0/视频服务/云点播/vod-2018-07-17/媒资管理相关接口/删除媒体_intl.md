## 1. API Description
API domain name: vod.tencentcloudapi.com.  
This API deletes media resource and its corresponding processed files, including transcoded videos, image sprites, screenshots, and videos published on WeChat.  
* The transcoded video files or video published on WeChat can be deleted individually with specified video ID.  
Default API request rate limit: 100 requests/sec. 

## 2. Input Parameters
The following parameters are required for requesting this API, including action-specific parameters and common parameters. For more information about common parameters for all requests, see [Common Request Parameters](/document/api/266/31756).

| Parameter name | Required | Type | Description |
|---------|---------|---------|---------|
| Action | Yes | String | Common parameter; the name of this API: DeleteMedia |
| Version | Yes | String | Common parameter; the version of this API: 2018-07-17 |
| Region | No | String | Common parameter; optional for this API |
| FileId | Yes | String | Unique ID of the media file. |
| DeleteParts.N | No | Array of [MediaDeleteItem](/document/api/266/31773#MediaDeleteItem) | Specify the ones to be deleted in the current call. The default value is "[]", which means to delete the media file and all its corresponding files generated by video processing. |
| SubAppId | No | Integer | ID of the VOD [sub-application](/document/product/266/14574). Input the ID of the sub-application that has the desired resources; otherwise, leave it blank. |
## 3. Output Parameters
| Parameter name | Type | Description |
|---------|---------|---------|
| RequestId | String | The ID of the request. Each request returns a unique ID. The RequestId is required to troubleshoot issues. |

## 4. Sample
### Sample 1. Deleting a Media File
The video file does not need to be retained, and all files and information under the FileId are to be deleted.
#### Input Sample Code
```
https://vod.tencentcloudapi.com/?Action=DeleteMedia
&FileId=7447398156998994860
&SubAppId=1
&<Common request parameter>
```
#### Output Sample Code
```
{
  "Response": {
    "RequestId": "requestId"
  }
}
```
### Sample 2. Deleting Transcoded Videos and Videos Published on WeChat
Delete all the transcoded video files and video files published on WeChat under the specified FileId.
Keep the source file and other files and information generated by other task flows such as transcoding and screencapturing.
#### Input Sample Code
```
https://vod.tencentcloudapi.com/?Action=DeleteMedia
&FileId=7447398156998994860
&SubAppId=1
&DeleteParts.0.Type=TranscodeFiles
&DeleteParts.1.Type=WechatPublishFiles
&<Common request parameter>
```
#### Output Sample Code
```
{
  "Response": {
    "RequestId": "requestId"
  }
}
```
### Sample 3. Deleting Certain Transcoded Videos
Delete transcoded videos in certain definitions and keep the rest ones.
For example, delete both the HLS file in the transcoding definition of 230 and the HLS Master Playlist.
#### Input Sample Code
```
https://vod.tencentcloudapi.com/?Action=DeleteMedia
&FileId=7447398156998994860
&SubAppId=1
&DeleteParts.0.Type=TranscodeFiles
&DeleteParts.0.Definition=230
&DeleteParts.1.Type=TranscodeFiles
&DeleteParts.1.Definition=10000
&<Common request parameter>
```
#### Output Sample Code
```
{
  "Response": {
    "RequestId": "requestId"
  }
}
```
