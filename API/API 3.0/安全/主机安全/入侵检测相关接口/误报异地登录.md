## 1. 接口描述

接口请求域名： yunjing.tencentcloudapi.com 。

本接口{MisAlarmNonlocalLoginPlaces}将设置当前地点为常用登录地。

默认接口请求频率限制：20次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见[公共请求参数](/document/api/296/19828)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：MisAlarmNonlocalLoginPlaces |
| Version | 是 | String | 公共参数，本接口取值：2018-02-28 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| Ids.N | 是 | Array of Integer | 异地登录事件Id数组。 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 误报异地登录

#### 输入示例

```
https://yunjing.tencentcloudapi.com/?Action=MisAlarmNonlocalLoginPlaces
&Ids.0=1
&Ids.1=2
&<公共请求参数>
```

#### 输出示例

```
{
  "Response": {
    "RequestId": "354f4ac3-8546-4516-8c8a-69e3ab73aa8a"
  }
}
```
