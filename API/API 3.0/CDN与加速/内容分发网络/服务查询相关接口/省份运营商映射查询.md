## 1. 接口描述

接口请求域名： cdn.tencentcloudapi.com 。

DescribeMapInfo 用于查询省份对应的 ID，运营商对应的 ID 信息。

默认接口请求频率限制：20次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见[公共请求参数](/document/api/228/30977)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：DescribeMapInfo |
| Version | 是 | String | 公共参数，本接口取值：2018-06-06 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| Name | 是 | String | 映射查询类别：<br/>ips：运营商映射查询<br/>district：省份映射查询 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| MapInfoList | Array of [MapInfo](/document/api/228/30987#MapInfo) | 映射关系数组。|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 查询省份/运营商映射关系

#### 输入示例

```
https://cdn.tencentcloudapi.com/?Action=DescribeMapInfo
&Name=isp
&<公共请求参数>
```

#### 输出示例

```
{
  "Response": {
    "RequestId": "fcd7aded-1866-467e-a9f6-d8d00b09557e",
    "MapInfoList": [
      {
        "Id": 3947,
        "Name": "中国铁通"
      }
    ]
  }
}
```
