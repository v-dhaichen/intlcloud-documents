## 1. 接口描述

接口请求域名： cam.tencentcloudapi.com 。

查询用户组详情

默认接口请求频率限制：20次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见 [公共请求参数](/document/api/598/33158)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：GetGroup |
| Version | 是 | String | 公共参数，本接口取值：2019-01-16 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| GroupId | 是 | Integer | 用户组 ID |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| GroupId | Integer | 用户组 ID|
| GroupName | String | 用户组名称|
| GroupNum | Integer | 用户组成员数量|
| Remark | String | 用户组描述|
| CreateTime | String | 用户组创建时间|
| UserInfo | Array of [GroupMemberInfo](/document/api/598/33167#GroupMemberInfo) | 用户组成员信息|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|
